# marked
Files that i use for my website, extends how `marked.js` works. These files might update at anytime without warning. I'll however try to not break everything during an update.

If you want a stable release version, then use the tag `latest` as the base.

## Installation
Clone
```sh
$ git clone git@github.com:dragmine149/marked.git
```
or submodule
```sh
$ git submodule add git@github.com:dragmine149/marked.git marked
```
this repo.

### External Installation
N/A. Currently the only way is via the repo. If you wish to use javascript instead, then see the `js` branch. If this gets enough attention that people want it more accessible. Then i will consider publishing it. Otherwise it's just another package
i don't want to maintain for the purpose of others.

Everything is documented in-file. If you don't know how to use [marked](https://marked.js.org/), go visit that first.
